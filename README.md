# 🏭 Industrial Copper Modeling

A Streamlit app that predicts:
- 💰 Selling Price (Regression)
- 📦 Lead Status (Classification)

### Built With:
- XGBoost
- Scikit-learn
- Streamlit
- Pandas, NumPy

### 🔮 Live Features:
- User-friendly form UI
- Realtime prediction
- Snow / Balloon animation on predict!
